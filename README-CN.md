# PasswordMemo

[English](./README.md)

## 🔑 隐私优先的纯前端密码管理器插件

PasswordMemo 是一款纯前端的浏览器插件式密码管理器。它设计轻巧、注重隐私，支持用户自定义数据模板，并可通过 WebDAV 或其他远程存储仓库进行加密同步。

-----

## 核心特性

1.  **跨平台支持：** 作为浏览器插件，支持桌面端和移动端浏览器，实现跨平台使用。
2.  **极简操作：** 操作直观简单，接近零配置。无任何复杂的后端服务或部署操作，即装即用。
3.  **高度自定义：** 支持用户自定义数据模板。除了传统的密码，还可以安全地保存 AI 服务的 API Key、长文本笔记等任意内容。
4.  **不接触用户数据：** 插件不接触任何用户数据。所有数据处理和存储都在本地完成，最大限度保障用户隐私。
5.  **精巧的安全实现：** 采用标准且精巧的加密实现方案，确保数据安全。
6.  **灵活的同步机制：** 利用用户自己的 WebDAV 服务器进行同步。通过 WebDAV 适配器，可实现对 Google Drive、OneDrive 等多种网盘的支持。（后续计划支持更多网盘和 GitHub 私有仓库作为存储）。

## 🛠️ 安装指南

### 浏览器商店安装

目前正在上架流程中，暂时还不支持通过商店直接安装。

### 通过 Releases 下载

请下载构建好的 `zip` 文件。由于尚未上架，可能需要在浏览器中开启**开发者模式 (Developer Mode)**，然后加载该 zip 文件进行安装。

### 自行构建

如果希望自行构建，请遵循以下步骤：

```bash
git clone https://github.com/gadingboding/PasswordMemo
cd PasswordMemo
pnpm install
pnpm run build
```

克隆并构建后，最终的未打包制品位于 `password-memo-browser/dist` 目录中。需要打开浏览器的**开发者模式**，然后使用 **“加载已解压的扩展程序 (Load unpacked)”** 方式来使用。

## 🗺️ 功能路线图

1.  **PassKey 解锁：** 利用操作系统的 PassKey，允许通过 PIN 或生物识别进行快速解锁，提升便利性。
2.  **WebAssembly 加密：** 将加密逻辑移植到 WebAssembly 实现，进一步提升安全性和性能。
3.  **AI 亲和性：** 增强对 AI 的支持，努力让不懂编程的用户能够通过 AI 快速实现自己想要的功能。
