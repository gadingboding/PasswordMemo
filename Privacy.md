# PasswordMemo Privacy Policy
## No Collection of User Data
This extension is designed as a pure frontend application and does not rely on any backend servers for storing, processing, or syncing user data.
It does not collect, store, or transmit any user’s personally identifiable information, browsing history, password data, or any other sensitive data to our servers or any third party.

## Data Storage and Synchronization
The data is stored in the browser's local storage and is protected by encryption. Users are free to configure a synchronization service for syncing the encrypted password file.
The user's data is transferred directly between the user's browser and the specified remote synchronization storage service. We are not involved in, do not intermediate, and do not possess the user's remote synchronization storage service credentials or the synced data. The security of data transfer and storage is the responsibility of the user and their chosen remote synchronization storage service provider.

## Permission Usage
All permissions used by this extension (such as webRequest, activeTab, etc.) are solely used to perform the extension's core functionality, including:
1. Establishing a secure connection to the user's configured remote synchronization storage service.
2. Retrieving the current tab's URL upon user action for context within the popup.
3. Opening settings or help documentation pages as requested by the user.
4. The use of all permissions is exclusively to support the extension's secure local operation and does not involve data collection or monitoring of user browsing activity.

# PasswordMemo 隐私政策
## 不收集用户数据
此扩展程序被设计为一个纯粹的前端应用程序，不依赖于任何后端服务器来存储、处理或同步用户数据。
此程序不会向我们的服务器或任何第三方收集、存储或传输任何用户的个人身份信息、浏览历史、密码数据或任何其他敏感数据。

## 数据存储和同步
数据被存储在浏览器本地存储，并被加密保护，用户允许自由配置同步服务，用于同步加密的密码文件。
用户的数据是直接在用户的浏览器和您指定的远程同步存储服务之间传输的。我们不参与、不介入，也不持有用户的远程同步存储服务凭证或同步的数据。 数据传输和存储的安全性是用户及其选择的远程同步存储服务提供商的责任。

## 权限使用
本扩展程序所使用的所有权限（例如 webRequest、activeTab 等）仅用于执行扩展程序的核心功能，包括：
1. 建立与用户配置的远程同步存储服务的安全连接。
2. 在用户操作时检索当前标签页的 URL，以便在弹出窗口中提供上下文。
3. 根据用户请求打开设置或帮助文档页面。
4. 所有权限的使用都专用于支持扩展程序的安全本地操作，不涉及数据收集或监控用户浏览活动。
